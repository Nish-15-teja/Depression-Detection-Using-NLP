# Depression Detection from Text Using Transformer Models with Sentiment-Lexicon Augmentation

## Overview

This project proposes SEEN-NLM, a hybrid deep learning framework that combines DistilBERT embeddings, VADER sentiment analysis, and a depression-specific lexicon to detect depressive text from social media posts.

The model captures both contextual meaning and emotional signals, improving depression detection compared to traditional NLP approaches.
